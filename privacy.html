Privacy Policy

Daily Kanji — Chrome Browser Extension
Last updated: April 2026

Summary: Daily Kanji does not collect, store, or transmit any personal data. The extension works entirely on your device.

1. Overview

Daily Kanji is a Chrome browser extension that displays one Japanese N5 kanji character each day, including its stroke order animation, readings, and meaning. This privacy policy explains how the extension handles your data.

2. Data We Do Not Collect

Daily Kanji does not collect any personal information. Specifically, the extension does not collect:

  • Personal identification information
  • Browsing history or activity
  • Location data
  • Device information
  • Usage statistics or analytics
  • Cookies or tracking data

3. Data Storage

Daily Kanji does not store any data on external servers. All kanji data is bundled locally within the extension and runs entirely on your device. No account creation is required.

4. External Requests

When you click the "Watch Stroke Order" button, the extension fetches an SVG file from the KanjiVG open-source repository hosted on GitHub (github.com/KanjiVG/kanjivg). This request contains no personal information — it is simply a file download of kanji stroke data.

Additionally, the extension loads the Noto Serif JP font from Google Fonts for display purposes. This is a standard web font request and does not transmit any personal data.

5. Third Party Services

Daily Kanji uses the following open-source resources:

  • KanjiVG — Open-source kanji stroke order data by Ulrich Apel, licensed under Creative Commons Attribution-Share Alike 3.0
  • Google Fonts — Noto Serif JP typeface for display

6. Children's Privacy

Daily Kanji is suitable for all ages and does not knowingly collect any information from children or any other users.

7. Changes to This Policy

If this privacy policy changes in the future, the updated version will be posted on this page with a revised date.

8. Contact

If you have any questions about this privacy policy, please open an issue on the GitHub repository.
